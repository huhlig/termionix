Struct AnsiCodec 

pub struct AnsiCodec<I> { /* private fields */ }
Expand description

ANSI codec for encoding and decoding ANSI sequences over a Telnet connection.

This codec wraps a TelnetCodec and uses an AnsiMapper to parse ANSI escape sequences from the byte stream. It implements both Decoder and Encoder traits from tokio_util for use with tokio’s framed I/O.
