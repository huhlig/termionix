Telnet Subnegotiation Argument

Variantsยง

ยง

NAWSWindowSize(WindowSize)

A subnegotiation for the window size, where the first value is the width and the second value is the height. The values are in characters.

ยง

CharsetRequest(Vec<BytesMut>)

Indicates an intent to begin CHARSET subnegotiation. This can only be sent after receiving a DO CHARSET after sending a WILL CHARSET (in any order).

ยง

CharsetAccepted(BytesMut)

Indicates that the receiver has accepted the charset request.

ยง

CharsetRejected

Indicates that the receiver acknowledges the charset request but will not use any of the requested characters.

ยง

CharsetTTableRejected

Indicates that the receiver acknowledges a TTABLE-IS message but is unable to handle it. This will terminate subnegotiation.

ยง

GMCP(GmcpMessage)

GMCP (Generic Mud Communication Protocol) message. Contains a package name and optional JSON data payload.

Encodes this TelnetArgument to a BufMut buffer.

Writes the argumentโs byte representation to a mutable buffer, automatically advancing the bufferโs write position. This is the preferred method for encoding into buffers that implement BufMut.

ยงArguments
  • dst - A mutable buffer implementing BufMut (e.g., BytesMut). The buffer is advanced by the bytes written.
ยงReturns

Returns Ok(bytes_written) on success, containing the exact number of bytes that were written to the buffer. Returns a CodecResult error if encoding fails.

ยงPerformance

This is an O(n) operation where n is the length of the encoded argument. It performs a single pass through the data, writing directly to the buffer without intermediate allocations.

ยงBuffer Behavior
  • The buffer is only written to; existing content is preserved
  • The bufferโs write position is advanced automatically
  • No bounds checking is performed; the buffer must have sufficient capacity
  • For BytesMut, automatic growth occurs if needed

use termionix_telnetcodec::naws::WindowSize;

pub fn option(&self) -> TelnetOption

Returns the TelnetOption associated with this argument.

Identifies which Telnet option this argument is for, enabling proper routing and handling of subnegotiation data during sidechannel processing.

ยงReturns

A TelnetOption enum value corresponding to the argumentโs option type.

ยงBehavior by Variant
  • NAWSWindowSize(_) โ Returns TelnetOption::NAWS

  • CharsetRejected โ Returns TelnetOption::Charset
  • CharsetTTableRejected โ Returns TelnetOption::Charset
  • Unknown(option, _) โ Returns the contained option
ยงPerformance

This is an O(1) operation that simply matches and returns a value.

ยงExamples
